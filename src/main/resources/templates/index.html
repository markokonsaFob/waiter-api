Welcome

I'm glad you have found your way here!

This project is created to give you some examples of API testing and API development.

Why are we doing this?

Nowadays clients are more and more in the mind that manual testers should have some experience in technical testing, including API testing.

We are trying to spread the word and find new quality assurance enthusiasts or back-end developers.

What can be done with this?

Right now we have created API which allows you to test your knowledge about API testing.

Root Endpoint

    http://localhost:8080/

Authentication

API is using sessionId parameter in header requests for authentication. You will find the sessionId token in login responses.

Register new user 
    Request type: POST

    http://localhost:8080/user/register

    Headers:
        sessionId

    Parameters:
        username
        password
        password2
        role


    Valid response:
        {
            "errorMessage": "",
            "response": {
                "id": "5aa66c0b76660a662e7acbc5",
                "username": "Marko",

            },
            "error": false
        }
